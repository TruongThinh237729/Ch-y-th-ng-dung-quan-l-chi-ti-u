📊 Phần Mềm Quản Lý Chi Tiêu Cá Nhân

Chào mừng thầy/cô đến với bản Demo đồ án Python Desktop.

🚀 TẢI BẢN DEMO (.ZIP)

📝 Hướng dẫn chạy chương trình:

  1. Tải file .zip về máy tính Windows.
  2. Giải nén thư mục (Chuột phải chọn Extract All).
  3. Nhấp đúp vào file QuanLyChiTieu.exe để bắt đầu sử dụng.
  4. Lưu ý: Nếu Windows báo bảo mật, chọn "More info" -> "Run anyway".
